
Streamlit Insurance / Attrition Dashboard

Contents:
- app.py : Main Streamlit application (run with `streamlit run app.py`)
- helpers.py : Preprocessing and modeling helper functions
- Insurance.csv : Sample dataset (demo). Replace with your own file if needed.
- requirements.txt : packages (no versions) as requested

How to use:
1. Create a new GitHub repo and upload all files at the root (no folders).
2. Connect the repo to Streamlit Cloud and set the main file to app.py.
3. Open the app on Streamlit Cloud. Use the 'Modeling' tab to train models (cv=5).
4. Use 'Predict & Upload' to upload new CSVs and download predictions.
